# RSA — Route Safety Analysis

Private road safety and route analysis platform for heavy transport operations.

## Monorepo structure

- `/backend`: Laravel 13 API
- `/frontend`: Next.js frontend
- `/docs`: architecture, decisions, testing and deployment documentation

## Current phase

RUP adapted phase: Inicio / Elaboración inicial.

First MVP goal:
- Route origin/destination input
- Google Maps route rendering
- Incident markers
- Incident sidebar
- Incident detail dialog
- Seeded incident data
- Laravel Reverb + Redis prepared for realtime updates
